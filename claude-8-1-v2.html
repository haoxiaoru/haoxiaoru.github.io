和弦的定义

和弦是同时演奏的三个或更多音符的组合，是西方音乐理论中的基本构成要素。和弦为音乐提供了和声结构，创造出丰富的音乐色彩和情感表达。

从理论上讲，和弦是根据特定的音程关系构建的。最基本的和弦是三和弦，由三个音符组成，这些音符按照特定的三度音程叠置而成。

基础和弦类型

和弦有多种类型，从最基本的三和弦到复杂的扩展和变化和弦。以下我们将分类介绍主要的和弦类型。

三和弦

三和弦是最基础的和弦类型，由三个音符组成，这些音符以三度音程叠加。根据音程结构的不同，三和弦可分为四种基本类型：

三和弦类型 结构 音程构成 例子（C为根音） 情感特点
大三和弦 (Major) 根音-大三度-纯五度 大三度+小三度 C-E-G 明亮、稳定、欢快
小三和弦 (Minor) 根音-小三度-纯五度 小三度+大三度 C-E♭-G 忧郁、柔和、深沉
减三和弦 (Diminished) 根音-小三度-减五度 小三度+小三度 C-E♭-G♭ 紧张、不稳定、神秘
增三和弦 (Augmented) 根音-大三度-增五度 大三度+大三度 C-E-G♯ 不安、梦幻、过渡性

七和弦

七和弦是在三和弦的基础上添加了一个七度音的和弦。根据三和弦的类型和添加的七度音的不同，七和弦可以分为多种类型：

七和弦类型 结构 例子（C为根音） 常见应用
大七和弦 (Major 7th) 大三和弦 + 大七度 C-E-G-B 爵士、流行音乐中的温暖和声
属七和弦 (Dominant 7th) 大三和弦 + 小七度 C-E-G-B♭ 古典和流行音乐中常见，有张力，倾向于解决到主和弦
小七和弦 (Minor 7th) 小三和弦 + 小七度 C-E♭-G-B♭ 爵士、R&B、放克中的柔和、丰富和声
半减七和弦 (Half-diminished 7th) 减三和弦 + 小七度 (m7♭5) C-E♭-G♭-B♭ 过渡和声，常用于小调的II-V-I进行
减七和弦 (Diminished 7th) 减三和弦 + 减七度 (dim7) C-E♭-G♭-B♭♭(等音为A) 强烈的不稳定感，对称结构，用于戏剧性转调或连接

扩展和弦

扩展和弦是在七和弦的基础上继续向上叠加三度音程（九度、十一度、十三度）而构成的和弦。这些和弦常见于爵士乐、融合音乐和现代流行音乐中，提供了更复杂、更丰富的和声色彩。

扩展和弦类型 构成 例子（C为根音） 说明
九和弦 (9th) 七和弦 + 九度音 Cmaj9: C-E-G-B-D
C9: C-E-G-B♭-D
Cm9: C-E♭-G-B♭-D		 在七和弦上增加色彩。
十一和弦 (11th) 九和弦 + 十一度音 C11: C-(E)-G-B♭-D-F
Cm11: C-E♭-G-B♭-D-F		 通常省略三音（避免与四度冲突）或五音。属十一和弦（C11）的11音常被视为sus4。
十三和弦 (13th) 十一和弦 + 十三度音 C13: C-E-G-B♭-(D)-(F)-A
Cmaj13: C-E-G-B-(D)-(F)-A
Cm13: C-E♭-G-B♭-(D)-(F)-A		 最复杂的扩展和弦。实践中常省略十一音、五音，有时甚至省略九音。

注意： 在实际演奏中，扩展和弦通常会省略一些音符（特别是内部音，如五音、有时是三音或十一音），以避免音响过于密集或产生不和谐的音程关系（如属和弦中的三音和十一音之间的大二度）。省略哪些音取决于和弦类型、音乐风格和声部进行。

变化和弦

变化和弦（Altered Chords）通常指变化属和弦（Altered Dominant Chords），它们是通过升高或降低属七和弦中的五音或/和九音（有时包括十一音/十三音）来创造更强烈的张力，以增强其解决到主和弦的倾向。这些和弦在爵士乐中尤为常见。

常见的变化包括：

  • 变化五音：
    • 增五音 (♯5 或 +5): 如 C7(♯5) = C-E-G♯-B♭
    • 减五音 (♭5): 如 C7(♭5) = C-E-G♭-B♭
  • 变化九音：
    • 增九音 (♯9): 如 C7(♯9) = C-E-G-B♭-D♯
    • 减九音 (♭9): 如 C7(♭9) = C-E-G-B♭-D♭
  • 组合变化： 可以同时变化五音和九音，例如：
    • C7(♭9♯5): C-E-G♯-B♭-D♭
    • C7(♯9♭5): C-E-G♭-B♭-D♯
    • C7alt (Altered Chord): 通常包含♭9, ♯9, ♭5, ♯5 中的一种或多种变化音，具体构成取决于上下文，常指包含♭9和♯9或♭9和♭5/♯5的和弦。

特殊和弦

除了基于三度叠加构建的标准和弦外，还有一些具有特殊结构或功能的和弦，在各种音乐风格中广泛使用：

特殊和弦类型 构成（以C为根音） 特点与应用
挂四和弦 (sus4) C-F-G (根音-纯四度-纯五度) 用四度音替代三度音，产生悬置感，通常解决回原来的大/小三和弦。常见于流行、摇滚、民间音乐。
挂二和弦 (sus2) C-D-G (根音-大二度-纯五度) 用大二度音替代三度音，声音明亮、开放，但不像sus4那样有强烈的解决倾向。
加六和弦 (add6 / 6) C6: C-E-G-A (大三和弦 + 大六度)
Cm6: C-E♭-G-A (小三和弦 + 大六度)		 在三和弦上添加大六度音（而非七度音）。C6声音温暖、略带复古感，常见于爵士、流行。Cm6声音复杂、略带神秘感，常见于爵士、电影配乐。
加九和弦 (add9) C(add9): C-E-G-D (大三和弦 + 大九度)
Cm(add9): C-E♭-G-D (小三和弦 + 大九度)		 在三和弦上添加九度音（而非七度音），为和弦增添色彩和亮度，但不改变其基本功能。
强力和弦 (Power Chord / 5) C5: C-G (根音 + 纯五度) 只有根音和五音（有时重复根音），省略三音。声音强劲、中性（无大/小调色彩），是摇滚和重金属音乐的基础。
四度和弦 (Quartal Chord) 例如：C-F-B♭ 或 E-A-D 基于四度音程叠加构建，而非三度。声音现代、空旷或不明确，常见于现代古典、爵士乐。
多重和弦 (Polychord) 例如：G大三和弦 / C大三和弦 同时演奏两个或多个不同的和弦。创造复杂、有时不和谐的音响，常见于现代音乐。

总结

和弦是音乐的基石，构成了和声的基础，赋予音乐深度、色彩和情感。从简单的三和弦到复杂的扩展和变化和弦，每种类型都有其独特的音响效果和功能。

和弦类型可以大致归纳为：

  1. 三和弦 (Triads): 大、小、增、减 - 构成和声的基本框架。
  2. 七和弦 (Seventh Chords): 在三和弦上加七度音，增加了复杂性和功能性（如属七的解决倾向）。主要有大七、属七、小七、半减七、减七。
  3. 扩展和弦 (Extended Chords): 在七和弦上继续叠加九、十一、十三度音，常见于爵士等需要丰富色彩的音乐。
  4. 变化和弦 (Altered Chords): 通常指变化属和弦，通过改变五音/九音增加张力。
  5. 挂留和弦 (Suspended Chords): Sus4 和 Sus2，用四度或二度代替三度，产生悬置感。
  6. 附加音和弦 (Added Note Chords): 如 add6, add9，在三和弦基础上添加音符而不构成七和弦。
  7. 其他特殊和弦： 如强力和弦 (Power Chords)，四度和弦，多重和弦等。

虽然基本和弦类型大约有几十种，但考虑到所有12个根音、不同的转位、省略音和细微变化，实际应用中的和弦组合数量极其庞大。理解和弦的构建原理、音响特性及其在和声进行中的作用，对于作曲、编曲、演奏和即兴都是至关重要的技能。

不断探索和实践各种和弦，是提升音乐理解和表达能力的关键途径。
