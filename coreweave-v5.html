报告摘要

CoreWeave, Inc.（股票代码：CRWV）是一家成立于2017年的美国AI云计算公司，总部位于新泽西州。公司最初以加密货币挖矿起家，后转型为利用其庞大的NVIDIA GPU资源提供AI云计算服务。CoreWeave现已成为北美最大的私人GPU提供商之一，专注于为AI、机器学习和深度学习应用提供高性能、高可扩展性的云基础设施。公司于2025年3月在纳斯达克上市，并凭借其在AI计算领域的领先技术和快速增长的市场份额，被《时代》杂志评为最具影响力的公司之一。

专注AI优化的云基础设施

一、公司概况

1.1 公司简介

CoreWeave, Inc.（股票代码：CRWV）是一家美国的AI云计算公司，成立于2017年，总部位于新泽西州利文斯顿 [27] 。公司最初以“Atlantic Crypto”的名义成立，主要从事加密货币挖矿业务，购买了大量的NVIDIA GPU用于挖矿和出租GPU服务器 [27] [59] 。2019年，公司更名为CoreWeave，利用其庞大的GPU资源，转型为提供AI云计算服务的公司 [27] [9] 。目前，CoreWeave是北美最大的私人GPU提供商 [59] ，专注于为AI工作负载提供高性能的云基础设施和服务。其平台专为AI、机器学习和深度学习应用而设计，能够提供高扩展性、高性能和高效率的计算资源 [11] [26] 。截至2024年，公司员工人数约为550人 [27] 。

1.2 发展历程

2017年

公司成立，最初名为Atlantic Crypto，从事以太坊挖矿业务，大量购买NVIDIA GPU [27] [9] 。

2019年

公司更名为CoreWeave，开始转型为AI云计算服务提供商，利用其GPU资源为其他公司提供计算服务 [27] [9] 。

2023年

完成B轮融资2.21亿美元，由Magnetar Capital领投，估值达到20亿美元 [31] 。成为首批提供NVIDIA H100 Tensor Core GPU云服务的提供商之一 [59] 。

2024年

被《时代》杂志评为最具影响力的公司之一，入选福布斯云100强 [33] 。收入达到19.2亿美元，同比增长730% [59] 。与微软签订多年GPU计算资源合同 [59] 。

2025年

3月在纳斯达克上市 (CRWV) [21] [66] 。Q1收入9.816亿美元，同比增长420% [49] [54] 。计划投资200-230亿美元扩展AI基础设施 [16] [46] [74] 。

二、业务与产品

2.1 核心产品与服务

CoreWeave的主要产品和服务包括 [11] [26] [33] ：

GPU云计算 (Compute): 提供高性能GPU计算资源，支持大规模AI模型训练和推理。基于Kubernetes原生环境，提供裸金属节点，实现最优性能 [12] [33] 。
存储 (Storage): 为AI工作负载优化的灵活存储解决方案，支持高速数据读写和大规模并行处理。
网络 (Networking): 高性能网络互联，采用NVIDIA InfiniBand技术，提供超低延迟和高带宽连接。
托管软件服务: 提供集群管理、性能监控等托管软件服务，减轻客户管理负担。
集群健康管理: 确保高可用性和可靠性的集群健康监测和管理服务。
平台服务: 提供从可观测性到安全性的全面平台服务，支持ML工具。

2.2 产品特点与优势

  • 专为AI优化的基础设施: 高性能、高扩展性、高效率 [11] [26] [33] 。
  • 大规模GPU集群: 超过25万块GPU，北美最大私人GPU集群之一 [27] [59] 。
  • 高性能网络技术: NVIDIA InfiniBand，超低延迟、高带宽 [12] [11] 。
  • 裸金属Kubernetes服务 (CKS): 托管Kubernetes，构建、训练、部署AI应用 [12] 。
  • 深度技术合作: 与NVIDIA等紧密合作，获取最新硬件技术 [27] [59] 。

三、市场表现与财务分析

3.1 市场表现

截至2025年5月16日，CoreWeave的股价为49.44美元（收盘价），市值约为221亿美元 [2] [3] [71] 。自2025年3月上市以来，股价经历一定波动，受市场对AI概念股热情及后续调整影响。

3.2 财务数据

收入 (Revenue):

  • 2024年全年：约19.2亿美元，同比增长730% [59] [31] 。
  • 2025年第一季度：9.816亿美元，同比增长420% [49] [54] 。

净亏损 (Net Loss):

  • 2024年全年：8.63亿美元 (2023年为5.94亿美元) [59] [31] 。
  • 2025年第一季度：3.146亿美元 [49] [54] 。

调整后EBITDA (Adjusted EBITDA):

  • 2024年全年：1.045亿美元 [54] [49] 。
  • 2025年第一季度：6.061亿美元 (利润率62%) [49] [54] 。

毛利率 (Gross Margin):

  • 2025年第一季度：62% [49] [54] 。

3.3 财务状况分析

  • 收入增长： AI市场需求旺盛及大客户合同驱动收入高速增长。
  • 亏损原因： 高资本支出、债务利息、股票期权等导致持续亏损。
  • 盈利能力： 调整后EBITDA显示经营层面盈利能力较强，毛利率高。
  • 债务压力： 截至2024年底总债务约80亿美元，偿债压力大 [31] [84] [85] 。
  • 现金流： 高资本支出和债务偿还导致自由现金流为负。

四、客户与合作伙伴

4.1 主要客户

微软 (Microsoft): 核心客户，占2024年收入的62% [31] [59] [36] 。为Azure云提供GPU计算资源。

OpenAI: 签订五年期、总价值119亿美元的AI基础设施合同 [16] [46] [74] 。

其他客户: 包括Meta等大型科技公司，以及众多AI实验室和企业 [6] [36] 。

4.2 合作伙伴

NVIDIA: 战略合作伙伴和投资者，提供先进GPU技术和支持 [59] [27] [70] 。

Run:ai: 2024年建立合作，提供AI工作负载和GPU编排平台 [80] 。

其他技术合作伙伴: 与多家软硬件公司合作，扩展服务生态系统。

五、行业地位与竞争分析

5.1 行业地位

CoreWeave被认为是AI云计算领域的领先者之一，专注于提供高性能、专为AI优化的基础设施 [69] [70] [33] 。

"CoreWeave在2024年被《时代》杂志评为最具影响力的公司之一，并入选福布斯云100强。"

— 综合公开信息 [33]

5.2 竞争对手

主要竞争对手包括 [41] [70] ：

  • 传统云服务提供商: 亚马逊AWS, 谷歌云, 微软Azure等。
  • 专业AI基础设施提供商: Lambda, Cerebras, Foundry, TensorWave等。
  • 其他竞争者: 传统HPC公司及新兴AI创业公司。

5.3 竞争优势

  • 专为AI优化的基础设施，性能效率高。
  • 大规模GPU资源 (>25万块)。
  • 技术创新，与NVIDIA等紧密合作。
  • 优质客户关系 (微软, OpenAI)。
  • 高性能裸金属Kubernetes服务 (CKS)。

5.4 竞争劣势

  • 客户集中度风险高。
  • 高债务和资本支出压力。
  • 品牌知名度相对较低。
  • 市场竞争加剧。

六、未来发展计划

6.1 扩张计划

  • 2025年投资计划: 投资200-230亿美元扩展AI基础设施和数据中心容量 [16] [46] [74] 。
  • 数据中心扩展: 在美国和欧洲扩展数据中心网络 [16] [46] [47] 。
  • 供应链多元化: 减少对单一供应商依赖，应对贸易紧张局势 [16] [46] 。

6.2 市场拓展

  • 拓展新客户群体: 减少对大客户依赖，降低客户集中度风险 [79] 。
  • 战略合作与并购: 如2025年初收购MLOps平台Weights & Biases [70] 。
  • 产品和服务创新: 持续研发，提升用户体验，满足多样化需求。

6.3 成本控制与财务目标

  • 优化资本支出: 提高运营效率，降低成本。
  • 提高盈利能力: 目标逐步实现盈利。
  • 债务管理与融资: 多种方式管理债务，降低财务负担 [84] 。

七、挑战与风险

  • 高债务压力: 2024年底总债务约80亿美元，偿债压力大，影响盈利能力 [31] [84] [85] 。
  • 客户集中度风险: 过度依赖微软、OpenAI等大客户，收入波动风险高 [31] [36] [79] 。
  • 高资本支出与盈利能力: 持续高额资本投入对现金流和盈利能力形成压力 [31] [85] 。
  • 技术更新风险: AI和云计算技术迭代快，面临技术落后和资产贬值风险 [70] [85] 。
  • 市场竞争风险: 传统云巨头和新兴AI公司竞争激烈，市场份额和利润承压 [41] [70] 。
  • 宏观经济与政策风险: 经济波动、利率、数据安全、国际贸易等政策法规变化可能带来不利影响 [64] [71] 。
